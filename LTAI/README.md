# Antalya Approach Version 0.1 (18C/L-WIP) 

This file contains LTAI's all SID/STAR procedures for runway 18C and 18L. 

It is possible to use 1D, 1S and 1X STARs. 

Area of responsibility has been extended beyond Antalya TMA. Covering outside border of Upper Sectors of Antalya APP and boundary of Gazipasa TMA. 

LTFG (Gazipasa) and LTFC (Isparta) will be added soon with 36s for LTAI.
